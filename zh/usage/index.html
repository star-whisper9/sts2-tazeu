快速入门 ​

安全须知

  • 务必在游玩前通过“测试电击”功能确认设定的强度在您的生理承受范围之内。
  • 请务必设置合理的“伤害上限”以防止高额伤害转换为过高电流。
  • 若感身体不适请立即切断连接或按下“断开连接”快捷键。
  • 本模组作者不对任何因使用本模组造成的身体不适或设备危险负责。

安装 ​

  1. (若已安装 ModConfig 请跳过此步骤) 前往 Nexus Mods (ModConfig) 下载最新版本的 ModConfig 模组。
  2. 前往 Nexus Mods 或 Github Releases 下载郊狼相生最新版本的发行包。
  3. 将发行包（ModConfig 与郊狼相生）解压到：
    • Windows：游戏根目录\mods\
    • macOS：游戏根目录 -> 右击 SlayTheSpire2.app 显示包内容 -> Contents/MacOS/mods/

TIP

mods 目录不存在请自行创建。

安装完成。

连接与使用 ​

  1. 打开游戏（若你是第一次使用 Mod，可能会提示你“是否要加载模组”，请点击“是”），进入游戏主界面后，服务将自动启动。确保游戏主界面右下角显示了“正在加载着模组运行。已加载 x 个模组。”的提示。

  1. 前往设置界面，打开“模组配置”选项，找到“TazeU”，至少为以下项设置快捷键：
    • 显示二维码
  2. 按下显示二维码按键，即可显示出能在郊狼 3.0 APP 扫码连接的二维码。

来到你的手机，确保手机与电脑在同一局域网（最简单就是连接了同一 WiFi），打开郊狼 3.0 APP，选择右下角的“SOCKET 控制”，然后进行扫码连接。连接成功后，二维码界面会显示绿色的已连接状态，并列出已连接的客户端。

WARNING

在郊狼 APP 中，请务必设置合理的强度上限！上限完全由 APP 端控制。

现在，Mod 就已经按照默认配置运行了，打开游戏来一把紧张刺激的电球鸡煲吧！
